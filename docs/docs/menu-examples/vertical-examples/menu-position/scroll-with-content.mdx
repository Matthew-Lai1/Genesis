# Scroll With Content

The `scrollWithContent` prop enables the navigation sidebar to be scrolled along with the page content. By default, the sidebar remains fixed and does not move when the page content is scrolled. However, when the `scrollWithContent` prop is set to `true`, the sidebar will scroll with the page content.

### Props

```ts noCopy
scrollWithContent?: boolean
```

```ts noCopy title='Default Value'
scrollWithContent={false}
```
