# Root Styles

export const updatedCode = SourceCode.replace(/customStyles=\{\{([\s\S]*?)\}\}/, '')

The Root Style feature provides the flexibility to apply custom styling to the navigation sidebar. With the `rootStyles` prop, you can pass a set of CSS properties to customize the appearance of the sidebar according to your design preferences. The `rootStyles` prop accepts a CSSObject from the [Emotion](https://emotion.sh/docs/introduction) library.

### Props

```ts noCopy
type RootStylesType = {
  rootStyles?: CSSObject
}
```
