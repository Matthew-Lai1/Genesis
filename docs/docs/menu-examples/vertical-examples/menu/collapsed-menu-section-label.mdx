# Collapsed Menu Section Label

When the menu is collapsed, the `collapsedMenuSectionLabel` prop can be used to display a custom label for each menu section. 

### Props

```ts noCopy
collapsedMenuSectionLabel?: ReactNode;
```

### Example

<BrowserWindow>
  <Provider>
    <CollapsedMenuSectionLabel />
  </Provider>
</BrowserWindow>
