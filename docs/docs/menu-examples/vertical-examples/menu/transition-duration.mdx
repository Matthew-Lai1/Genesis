The `transitionDuration` prop allows you to customize the duration of the animation when opening or closing submenus. The value is in milliseconds (ms).

### Props

```ts noCopy
transitionDuration?: number
```

```ts noCopy title='Default Value'
transitionDuration = 300; // milliseconds
```
