# Default Open

The `defaultOpen` prop is used to specify whether a submenu should be open by default when the page loads. By setting the `defaultOpen` prop to true, you can indicate that a specific submenu should be initially open.

### Props

```ts noCopy
defaultOpen?: boolean
```

```ts noCopy title='Default Value'
defaultOpen={false}
```
