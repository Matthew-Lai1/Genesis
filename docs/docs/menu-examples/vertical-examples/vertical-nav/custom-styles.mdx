# Custom Style

The Custom Style feature provides the flexibility to apply custom styling to the navigation sidebar. With the `customStyles` prop, you can pass a set of CSS properties to customize the appearance of the sidebar according to your design preferences. The `customStyles` prop accepts a CSSObject from the [Emotion](https://emotion.sh/docs/introduction) library.

### Props

```ts noCopy
customStyles?: CSSObject
```
