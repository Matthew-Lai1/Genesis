# Width

The `width` prop allows you to customize the width of the navigation sidebar according to your needs. You can set it to a number value, which represents the width in pixels.

### Props

```ts noCopy
width?: number
```

```ts noCopy title='Default Value'
width={260}
```

### Example

  <Provider>
    <Width />
  </Provider>
</BrowserWindow>
